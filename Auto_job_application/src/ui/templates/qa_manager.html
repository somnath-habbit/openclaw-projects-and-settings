How It Works
  • When applying via Easy Apply, the bot matches form questions against this database
  • If a match is found, the stored answer is used automatically
  • The "Used" count shows how many times each answer has been reused
  • Use "Quick Update" to change CTC/Notice Period across all related questions at once
  • For numeric fields (CTC in LPA, Notice Period in days), use just the number (e.g., "60" not "60 LPA")
